## 2 - Services sección - Reutilizable

1. Sección reutilizable en las siguientes plantillas: Inicio, colecciones, contenido, producto, blog
2. Todas las configuraciones deben ser compatibles con metacampos (usar contenido dinámico) cuando se utilicen en páginas de producto/colección (evitar área de texto, casillas de verificación, rango, etc.)
3. ID administrable (se recomienda configuración de texto): este ID se puede agregar a la URL de la página y usarse como ancla
  1. Ejemplo: [ornevo.fr](http://ornevo.fr)#\[id-sección\] desplazar para que la sección sea visible en la pantalla
  2. Recuerde tener en cuenta el nivel de desplazamiento y la altura del menú para que el menú no oculte el contenido
4. **Configuración de la sección**
  1. ID de sección
  2. Titulo de la sección -
  3. Subtitulo de la sección
5. **Cada bloque es un tab o pestaña - Configuración de cada pestaña - Max 4 bloques**
  1. Título administrable
    1. Hn administrable
  2. Texto enriquecido administrable, máximo 1 líneas en escritorio, max 3 líneas (en móvil)
  3. Icon SVG para el número que aparece en el tab
  4. Imagen administrable con ratio 1:1 para el circulo grande
  5. Lado derecho
    1. Titulo administrable, 1 línea max en desktop, 2 lineas max en móvil
    2. Texto enriquecido administrable, máximo 1 líneas en escritorio, max 3 líneas (en móvil)
    3. Servicios a 2 columnas, Max grid 2×4, osea 8 elementos max, en Responsive una columna
      1. Para cada servicio
        1. llenar primero columna 1
        2. Icono SVG administrable ratio 1:1
        3. Texto enriquecido max 2 lineas en desktop, max 2 lineas en responsive
        4. CTA administrable (opcional)
          1. Texto administrable
          2. URL administrable con opción para abrir en la misma pestaña o en una nueva
            1. Importante incluir una opción para abrir el formulario de contacto directamente
          3. Icono en SVG administrable
          4. Configuración para elegir entre el estilo de CTA del sistema de diseño (lista desplegable)
            1. Estilo 1
            2. Estilo 2
